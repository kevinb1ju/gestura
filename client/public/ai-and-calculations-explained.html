🧮 How Calculations Are Done

1. 🎮 Game Performance Tracking

Each game tracks multiple metrics during gameplay:

// Cognitive Domain Calculations calculateCognitiveScores(accuracy, events) { return { pattern_recognition: Math.min(100, accuracy * 1.1), attention_span: Math.min(100, (cognitiveEvents.length / 10) * 100), problem_solving: Math.min(100, (events.filter(e => e.type === 'problem_solved').length / 5) * 100), memory: Math.min(100, (events.filter(e => e.type === 'memory').length / 3) * 100) }; }
// Motor Domain Calculations calculateMotorScores(timeInSeconds, events) { return { hand_eye_coordination: Math.min(100, (motorEvents.length / timeInSeconds) * 10), fine_motor_control: Math.min(100, (events.filter(e => e.type === 'drag').length / 5) * 100), reaction_time: Math.min(100, 100 - (timeInSeconds / 60) * 10) }; }

2. 📈 Developmental Domain Scoring

Performance is calculated across four developmental domains:

🧠 Cognitive

  • Pattern Recognition: accuracy × 1.1
  • Attention Span: (events ÷ 10) × 100
  • Problem Solving: (solved ÷ 5) × 100
  • Memory: (memory events ÷ 3) × 100

🤸 Motor

  • Hand-Eye Coordination: (events ÷ time) × 10
  • Fine Motor Control: (drag events ÷ 5) × 100
  • Reaction Time: 100 - (time ÷ 60) × 10

👥 Social

  • Cooperation: (social events ÷ 3) × 100
  • Communication: (communication ÷ 2) × 100
  • Sharing: (sharing events ÷ 2) × 100

😊 Emotional

  • Frustration Tolerance: 100 - (frustration × 10)
  • Persistence: (persistence events ÷ 3) × 100
  • Confidence: accuracy score
  • Emotional Regulation: (duration ÷ 300) × 100

3. 📊 Overall Performance Calculation

// Calculate overall domain scores calculateDomainScores(performanceData) { const domainScores = { cognitive: 0, motor: 0, social: 0, emotional: 0 }; let gameCount = 0; Object.values(performanceData).forEach(gameData => { if (gameData.cognitive) { domainScores.cognitive += Object.values(gameData.cognitive).reduce((a, b) => a + b, 0) / 4; } if (gameData.motor) { domainScores.motor += Object.values(gameData.motor).reduce((a, b) => a + b, 0) / 3; } // ... similar for social and emotional gameCount++; }); // Average across all games Object.keys(domainScores).forEach(domain => { domainScores[domain] = domainScores[domain] / gameCount; }); return domainScores; }
