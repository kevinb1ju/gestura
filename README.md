# Gestura

signA comprehensive web application designed to provide an intuitive user interface with robust backend functionality. Gestura offers a seamless experience for users to manage and interact with content through a modern, responsive desin.

## Key Features

- **User Authentication**: Secure login and registration system
- **Admin Dashboard**: Comprehensive interface for content management
- **Responsive Design**: Works seamlessly across all devices
- **Real-time Updates**: Live data synchronization
- **Secure API**: Protected endpoints with JWT authentication

## Project Structure

```
Gestura/
├── client/          # Frontend React application
└── server/          # Backend Node.js server
```

## Prerequisites

- Node.js (v14 or later)
- npm (v6 or later) or yarn

## Getting Started

### Backend Setup

1. Navigate to the server directory:
   ```bash
   cd server
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

### Frontend Setup

1. Navigate to the client directory:
   ```bash
   cd client
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Available Scripts

In both client and server directories, you can run:

- `npm start` - Starts the development server
- `npm test` - Launches the test runner
- `npm run build` - Builds the app for production
